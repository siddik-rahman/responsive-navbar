# Responsive Navbar

A simple and responsive navigation bar built using HTML, CSS and JavaScript.  
This project is designed to work seamlessly on both mobile and desktop devices.

---

## 📌 Features

- 📱 **Responsive Design** – Works on all screen sizes (mobile, tablet, desktop)
- 🔗 **Navigation Links** – Users can click links to navigate to different sections
- 🧠 **Simple & Clean UI** – Minimalist and user-friendly interface

---

## 🛠️ Technologies Used

- HTML  
- CSS (Media Queries for responsiveness)  
- JavaScript (for toggling mobile menu)

---

## 🚀 How to Use

Just open the `index.html` file in any modern browser.  
No installation required.
